# Airbnb Data Analysis (Power BI)

## Overview

This project analyzes global Airbnb data to identify trends in listings growth, customer engagement, pricing, and city-wise performance. The goal is to leverage data insights to understand market dynamics, user behavior, and support better business decision-making.

---

## Dataset

The dataset is sourced from Maven Analytics Data Playground:
[https://mavenanalytics.io/data-playground/airbnb-listings-reviews](https://mavenanalytics.io/data-playground/airbnb-listings-reviews)

It includes information on:

* Listings data (property type, price, location)
* Reviews data (frequency, trends, customer engagement)
* Host information
* City-wise performance metrics
* Property categories (entire place, private room, shared room)

---

## Tools & Technologies

* Power BI – Data visualization and dashboard creation
* DAX – Calculated measures and KPIs
* Data Modeling – Relationships between datasets

---

## Steps Performed

### 1. Data Preparation

* Cleaned and transformed raw data
* Handled missing values
* Converted data types (date, numeric fields)

### 2. Data Modeling & Analysis

* Created relationships between listings and reviews
* Built DAX measures for KPIs (listings, reviews, pricing)
* Analyzed trends across cities, property types, and time

### 3. Data Visualization

* Designed an interactive Power BI dashboard
* Visualized listings growth and review trends
* Compared city-wise and property-wise performance

---

## Dashboard Highlights

* Listings growth over time
* City-wise market share analysis
* Property type and pricing comparison
* Review trends and seasonal patterns
* Trust analysis (verified vs non-verified users)

---

## Key Results

* Listings grew rapidly until 2015, followed by fluctuations due to regulations and COVID
* Major cities like Paris, New York, and Sydney dominate listings and reviews
* Review activity peaked in 2019 and declined after 2020
* Seasonal trends influence customer activity and demand
* Verified users form the majority, indicating higher platform trust

---

## Conclusion

This project demonstrates how data analytics and visualization can be used to understand platform growth, customer behavior, and market trends. The insights derived can help improve strategic decision-making in the travel and hospitality industry.
